# Crime Pre-processing and EDA instructions

This README includes instructions on how to run the Crime-Data-Preprocessing&Cleaning and Crime-EDA Jupyter notebooks.

## Getting Started
When downloading the submission, file positions should not be changed. The location of the top folder can be decided by the user.
### Prerequisites - Imports:
- os
- glob
- pandas
- numpy
- geopandas
- shapely
- matplotlib
- seaborn
- xml
- matplotx

Any missing libraries, using !pip install, installing the latest versions will suffice.

### Executing
1. Run Crime-Data-Preprocessing&Cleaning to achieve the cleaned data
2. Run Crime-EDA
